HTML


설계

p tag : 이 태그를 이용하여 리스트에 글, 내용을 적어 출력하게 하였습니다.

ol, li tag : 리스트를 정의하고 요소를 생성하게 합니다. 해당 태그는 각 페이지를 구분지을 수 있게 하며 좌측에 위치하고 있습니다. (개요, 개발일지, Diagram, 분석, HTML, CSS)

img, str, u tag : img는 이미지를 출력하기 위해 사용하며 '개요', 'index.html', '분석' 페이지에 사용되었습니다. str은 글자를 진하게 표현하며 u는 해당 글자에 밑줄을 표기합니다.

iframe tag : iframe은 인라인 프레임을 정의할 때 사용합니다. 즉 현재 파일에 다른 파일을 포함시켜야 할때 사용하게 됩니다. '개요' 항목에 유튜브에 있는 생활코딩 동영상을 가져오는데 사용하였습니다.

link tag : link는 외부 파일을 열려있는 파일에 적용하기 위해 사용합니다. 저는 css style 부분에 중복을 줄이기 위해 css파일을 따로 생성하고 link로 연결하였습니다.

table tag : table은 표를 표현하기 위해 사용합니다. 이 태그를 사용하기 위해 tr, th, td를 추가해야 합니다. tr은 표의 행을 th는 열의 제목 td는 표의 셀을 정의합니다. 이는 '개요'에 구현하였으며 언어 습득 현황판으로 이용하고 있습니다.

form tag : form은 사용자로 부터 데이터를 입력받을 수 있는 입력 부분을 의미합니다. 이 사이트에서 적용된 것은 dropdown list이며 'Diagram'에 존재합니다.
